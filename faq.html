What is VoiceApp?

VoiceApp is a push-to-talk dictation app for macOS. You hold a hotkey, speak, and release — and your words are typed into whatever app your cursor is in: an editor, browser, chat box, or terminal. It works everywhere the keyboard works, so you're not limited to one app.

Is VoiceApp private? Does my voice leave my Mac?

It depends on the model you choose. With the built-in on-device models (Whisper and Parakeet), transcription runs entirely on your Mac and no audio leaves the device. If you opt into a cloud provider (Grok, OpenAI, or ElevenLabs) using your own API key, audio is sent to that provider for transcription. On-device mode is the default for privacy.

What's the difference between the on-device and cloud models?

On-device models (Whisper, Parakeet) run locally, work offline, are private, and are free to run. Cloud models (Grok, OpenAI gpt-4o-transcribe, ElevenLabs Scribe) can be more accurate on hard audio and support more languages, but require your own API key and send audio to that provider. You can switch between them at any time in Settings.

Which languages does VoiceApp support?

The Whisper on-device models and most cloud providers support 90+ languages with automatic language detection. The Parakeet model is English-only but very fast. You can set a specific language or leave it on auto-detect.

What are the system requirements?

VoiceApp runs on Apple Silicon Macs (M1 or newer) on a recent version of macOS. It's a small download, and the on-device speech models download once and then work offline.

How do I set up VoiceApp?

Download the app, open it, and grant Microphone and Accessibility permissions when prompted (Accessibility lets it type into other apps). Pick a hotkey, choose a model, and start dictating. There's no account to create.

Does VoiceApp work in any app?

Yes. Because VoiceApp types your transcription as keyboard input, it works in any Mac app that accepts text — email, documents, browsers, chat apps, code editors, and the terminal.

Is VoiceApp better than Apple's built-in dictation?

VoiceApp offers a choice of modern speech models (including large Whisper and cloud AI), a consistent push-to-talk hotkey that works the same in every app, custom vocabulary, and automatic updates. Apple Dictation is built in and convenient, but VoiceApp gives more control over accuracy, models, and privacy. See the comparison for details.

How do I update VoiceApp?

VoiceApp checks for updates automatically and can install them in place. Updates are code-signed and notarized by Apple, so they install cleanly without security warnings.

How much does VoiceApp cost?

VoiceApp is free to download and try — see the download page for the latest on pricing. The on-device models are free to run; cloud models use your own API key and are billed by that provider.
